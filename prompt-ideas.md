Voici des suggestions de prompts utiles pour un projet de jeu :

  Développement

  | Commande    | Description                                                        |
  |-------------|--------------------------------------------------------------------|
  | /add-weapon | Guide pour ajouter une nouvelle arme (stats, sprites, intégration) |
  | /add-enemy  | Template pour créer un nouveau type d'ennemi avec son AI           |
  | /add-scene  | Scaffolding d'une nouvelle scène Phaser                            |
  | /add-item   | Ajouter un item/consommable au système d'inventaire                |

  Qualité de code

  | Commande   | Description                                            |
  |------------|--------------------------------------------------------|
  | /review    | Review un fichier/système et suggère des améliorations |
  | /find-debt | Identifie la dette technique dans une zone du code     |
  | /refactor  | Propose un plan de refactoring pour un système         |

  Debugging

  | Commande      | Description                                                      |
  |---------------|------------------------------------------------------------------|
  | /trace-bug    | Analyse un comportement inattendu et trouve la cause             |
  | /explain-flow | Explique le flux d'exécution d'une action (ex: "tirer une arme") |

  Testing

  | Commande        | Description                                          |
  |-----------------|------------------------------------------------------|
  | /generate-tests | Génère des tests pour un système/fichier             |
  | /test-coverage  | Analyse la couverture et suggère des tests manquants |

  Game Design

  | Commande            | Description                                              |
  |---------------------|----------------------------------------------------------|
  | /balance-check      | Analyse l'équilibrage (DPS, TTK, courbes de progression) |
  | /playtest-checklist | Génère une checklist de test pour une feature
